# Smart AI Examination Seating Arrangement System

Frontend-only React application for demonstrating an AI-inspired college examination seating dashboard, room visualization, roll-number search, and explainable seating traces.

## Stack

- Frontend: React + Vite
- Data source: local demo data in `frontend/src/demoData.js`
- UX modules: seating dashboard, room visualization, search metrics, reasoning trace, roll-number lookup

## Frontend setup

```bash
cd frontend
npm install
npm run dev
```

The app runs entirely in the browser and does not require a backend server.

## Key capabilities

- Frontend-only seating plan demonstration
- Multi-classroom seat visualization
- Roll number seat lookup
- Search algorithm metric cards
- Explainable AI-style reasoning traces
- Responsive admin-style dashboard
